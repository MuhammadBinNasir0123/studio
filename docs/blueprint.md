# **App Name**: PetPal

## Core Features:

- User Authentication: Secure registration and login with email/password and Google authentication using Firebase Authentication.
- Pet Management: Add, update, and manage pet profiles with details like name, breed, age, weight, and photo, stored in Firebase Firestore.
- Health Tracking: Record and track pet health records including vaccinations, vet visits, and medications, stored in Firebase Firestore.
- Reminder System: Set and manage reminders for feeding, walking, and medication schedules with push notifications via Firebase Cloud Messaging.
- Activity Logging: Log daily activities such as meals, walks, and grooming, and store the data in Firebase Firestore.
- Statistics and History: View statistics and history for each petâ€™s activities and health records, retrieved from Firebase Firestore.
- Profile Sharing (Optional): Allow users to share pet profiles or achievements with friends.

## Style Guidelines:

- Primary color: Soft teal (#4DB6AC) to convey a sense of calm and health.
- Secondary color: Warm orange (#FFB74D) to represent energy and playfulness.
- Accent color: Light blue (#64B5F6) for interactive elements and notifications.
- Body font: 'Roboto' for readability and a modern feel.
- Headline font: 'Montserrat' for clear and engaging titles.
- Use simple, clear icons from Material Design for easy recognition of features and actions.
- Clean and intuitive layout with a bottom navigation bar for primary features.
- Card-based design for pet profiles and activity logs to organize information effectively.
- Subtle animations for transitions and loading states to enhance user experience.
