# MyPetcare App

**MyPetcare App** is a comprehensive pet management application built with **Firebase Studio**, designed to help pet owners track and manage all aspects of their pets’ lives in one place. The app allows users to create profiles for multiple pets, record health information such as vaccinations, medications, and vet visits, log daily activities like meals, walks, and grooming, and receive **push notifications** for reminders including feeding, medication, and appointments.

The app features a **secure authentication system** via Email or Google login, a **mobile-friendly responsive interface**, and integrates with **Firebase Firestore** for database management and **Firebase Cloud Messaging** for notifications. It leverages modern web technologies including **HTML, CSS, JavaScript, TypeScript**, and **Tailwind CSS** for a smooth and interactive user experience.

### Key Features

* Multi-pet support with detailed profiles (name, breed, age, weight, and photo)
* Health tracking including vaccinations, medications, and vet visits
* Daily activity logging for meals, walks, grooming, and exercise
* Timely reminders and push notifications
* Clean, responsive design optimized for mobile

### Tech Stack

* **Frontend:** HTML, CSS, JavaScript, TypeScript, Tailwind CSS
* **Backend & Database:** Firebase Firestore
* **Authentication:** Firebase Authentication
* **Notifications:** Firebase Cloud Messaging
* **Hosting:** Firebase Hosting

### Purpose & Impact

**MyPetcare App** is a beginner-friendly full-stack Firebase project demonstrating practical, real-world functionality. It empowers pet owners to stay organized, monitor pet health, log activities, and get timely reminders, all through a simple, intuitive, and responsive interface.
